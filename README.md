# Scholar Awards Frontend PWA

## Quick start
```bash
npm install
cp .env.example .env
npm run dev
```

## Included
- role-wise dashboard shell
- students page
- category page with anchor mapping
- stage assignment page
- in-app live notifications
- system-flow page
- PWA manifest and service worker stub

## Not fully complete yet
- IndexedDB offline cache
- push notification permissions / subscription
- certificate editor canvas
- actual WhatsApp automation calls
- fine-grained page hiding by permission
